# 两类不同的 LLM

![nof1ai](./img/nof1ai.png)

上图是目前性能相对最强大的 6 个 LLM 进行的一场持续性地 crypto 市场永续合约量化投资竞赛。截至撰写本文时，该竞赛已经持续超过一周的交易时间，从图中可以明显观察出 6 个模型分为非常明显的 3 个收益率分段。而这一周的 BTCUSD 价格走势如下图所示。

![BTCUSD](./img/BTCUSD.png)

那么是哪些主要原因导致不同 LLM 出现如此巨大的结果差距呢？这里仅从幻觉（Hallucination）这个角度分析。

对于 LLM 来说，执行不同任务时，产生幻觉可能是有利也可能是不利的：

- 对于执行增强型搜索任务和推理任务的 LLM，确保搜索的准确性和逻辑的一致性是最重要的，此时如果产生幻觉，将会是巨大的错误。
- 对于执行创意扩展任务的 LLM，尽可能增加对所有可能结果的生成是最重要的，此时如果产生幻觉，甚至有可能是灵光一现。

落实到本次不同 LLM 的量化结果分析，由于这一周的 crypto 市场价格走势处于长时间震荡并在后期出现价格突破收敛区间的环境，

- 对于 DEEPSEEK CHAT V3.1 和 QWEN 3 MAX 这样的 LLM，可以将其视为非常有市场直觉优势的交易员，能够准确把握市场的节奏。
- 对于 GPT 5 和 GEMINI 2.5 PRO 这样的 LLM，可以将其视为非常保守的交易员，频繁地执行止损使其错失了当市场机会来临时修复净值的时机。

因此可以明显推断出前者这类 LLM 属于幻觉有利模型，适用于执行创意扩展任务；后者这类 LLM 属于幻觉不利模型，适用于执行增强型搜索任务和推理任务。

总结，利用具有不同优势的 LLM，使其能够协同配合，将能更好地执行复杂的任务并生成更佳的结果。
