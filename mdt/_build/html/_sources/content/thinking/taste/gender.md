# 两性 Gender

## 女性

- 端庄大气

符合该形象特征的女性，容易让人联想到贤能，拥有和配偶一起同甘共苦的性格特质，属于反脆弱的气质类型。对于男性来说，符合该形象特征和性格特征的女性是成就大业不可或缺的另一半。

- 甜美活力

符合该形象特征的女性，容易让人联想到青春，能带给配偶身心上的动力。但是由于这种形象与气质类型极易随着时间的流逝而消失，属于脆弱的气质类型。对于男性来说，符合该形象特征和性格特征的女性适合成为功成名就之后全身而退享受生活时的伴侣。

- 明艳娇媚

符合该形象特征的女性，容易让人联想到红玫瑰，美丽四射容易招蜂引蝶，属于反脆弱的气质类型。对于男性来说，符合该形象特征和性格特征的女性，容易招致风险因此不适宜作为配偶，但适合作为合作伙伴协同推进，不过依然不可交付信任。

- 清冷温婉

符合该形象特征的女性，容易让人联想到白玫瑰，遇良人会被怜爱，遇恶人会被摧残，属于脆弱的气质类型。对于男性来说，符合该形象特征和性格特征的女性，如果男性作为良人会被分散精力，如果男性作为恶人也只是将其利用，无论立场如何都不适宜长期相处。

## 男性

- 海盗

符合该性格特征的男性，追寻的是自由，虽然日常稳定性较弱，但是遭遇极端风险时的抵抗能力较强，属于反脆弱的气质类型。

- 教授

符合该性格特征的男性，追寻的是智慧，虽然日常平淡，但随着时间的推移，智慧所提供的抵御风险的保障越来越强大，属于反脆弱的气质类型。

- 狼人

符合该性格特征的男性，追寻的是强健，从精神上的刚强到身体上的健壮，可以为身边人面对风险时提供保护，但有随着时间推移而逐渐丧失强健气质的可能性，属于脆弱的气质类型。

- 吸血鬼

符合该性格特征的男性，追寻的是高贵，该气质只能源于家庭优渥环境的培养，且必须使用上乘的资源时刻进行维护，有面临极端风险而戛然而止的可能性，属于脆弱的气质类型。
